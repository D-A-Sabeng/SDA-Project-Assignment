# Time Use Survey - Finland

### Statistical Data Analysis Course Project Assignment

Preamble to the task 
As you are aware, we recently launched a survey to gather information about the habits of individual persons and households in Finland. The survey data is now available for analysis. Your task is to analyse which activities people spend their time on and whether there are differences between groups. ........ See. habits.txt file for more details about the project and documentation on data

This project was completed using Google Colab and demonstrates data analysis and modelling in Python.


## Files
- Analysis Code.ipynb - Main Notebook
- habits.data - dataset used for the assignment
- habits.txt - Assignment instruction and documentation about the datasets
- Project Final Report.pdf - The Final Report of the Assignment


## Tools
- Python
- pandas
- numpy
- sklearn
- scipy.stats
- matplotlib
- statsmodels
- scikit-learn


## How to run

Open the notebook in Jupyter or Google Colab
